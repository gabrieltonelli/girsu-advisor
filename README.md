# GIRSU Advisor - App Móvil Cliente

Supervisor de puntos de recolección

## Descripción

Aplicación móvil cliente para personal supervisor de puntos de recolección de la gestión de residos sólidos urbanos.
Permite relevar cada locación de contenedores y campanas de residuos de la via pública para realizar el informe automático del estado de las mismas.

## Utilización

1) El usuario ingresa, por única vez, con las credenciales que le fueron otorgadas (previamente generadas desde el panel de control). Para las veces posteriores bastará con que el usuario inicie la app, ya no se le solicitarán las credenciales. 

2) En cada locación de su recorrido, donde se hallan situados los contenedores y/o campanas de residuo, el usuario realiza la toma de una foto. Puede ingresar un comentario adicional si lo desea.

3) Envía el registro

## Registro

La aplicación cliente sube al servidor la información del relevamiento efectuado, el cual constará de, fotografía capturada, datos del usuario, fecha y hora de registro, coordenadas de latitud y longitud (identificación del contenedor o campana de residuo) y comentarios extras los cuales son opcionales.
